Why a remote conference?

You need to keep learning. One of the most rewarding ways to do that is by attending conferences. Unfortunately, sometimes it's difficult to make it to a conference due to travel costs, the price of the conference ticket, or your ability to take time off.

So, clear your calendar, grab some popcorn and twizzlers, and join us for some great content around building web and mobile apps. Or, grab a Users' Group ticket, get a company to provide a projector and pizza, and learn more realtime web development with your friends..

Using GraphQL As The Core Data Layer

Arunoda Susiripala

GraphQL is an application level query language from Facebook. At Kadira, we use GraphQL as the core data layer and it's used by almost every microservice in Kadira. We also used it to power charts in Kadira UI as well. In this talk, we'll focus why we use GraphQL and the process we used migrate our services to utilize GraphQL.

5 Tips To Improve Your JavaScript, with ES6

Derick Bailey

In this session, then, you'll see 5 tips to help improve your JavaScript and your browser code with the latest version of the language, ES6 (aka ES2015). You'll see what it takes to run ES6 code today, and the tools you need to support the features you want in browsers that aren't yet up to date.

By the time this session is done, you'll have the knowledge you need to get in, get out, and get back to doing what you love.

NoSQL Injection in Modern Web Applications

Pete Corey

The threat of SQL injection in modern web applications has been left by the wayside with the rise of NoSQL databases. Unfortunately, a new, but fundamentally similar threat has taken its place: NoSQL injection. Let's take an in-depth look at this type of attack and the steps we can take to protect ourselves from it. The king is dead, long live the king!

Delivering High Quality Features, Faster

Sam Hatoum

When you first start writing your app, you and your team will typically go at lightning speed. But as your application and its complexity grow and as your team increases in size, you start to feel the scaling pains and your productivity will be decreased through a combination of defects and regressions. Sam Hatoum will show you how you can mitigate this risk by putting quality first and building it in to your process.

Building Content-Oriented Sites with React

Karl Danninger

Content-oriented sites in Javascript frameworks should be fun! Karl is a full-stack developer at OK GROW! and will walk through how to solve some of the more interesting problems when building content driven sites in Javascript. Karl talks about Server Side Rendering and SEO, while serving assets from a CDN. Autoscrolling and saving the scroll position between pages, and a simple and easy way to create and edit content throughout your site with React & Meteor.

Using React and CSS Modules to make super fancy reusable components

Ramsay Lanier

Inline styling with React is for the birds. Ramsay will show you how to create React components with locally scoped CSS by leveraging ES2015 modules inside of a application built with Webpack.

Writing DDP APIs to Link Realtime Apps

Ian Serlin

The realtime capability of our apps often halts abruptly at the edge of our own codebase. Ian Serlin, CEO of Useful IO, thinks we can do better. Get a glimpse at how their team built Hail (a realtime communication service) and the simple steps you can take to create and publish realtime APIs to the world.

Finding the Holy Grail of MVP Stacks

Ben Strahan

Join Ben Strahan (Full-stack developer at OK Grow) as he takes you on his journey to find the ultimate stack for building minimal viable mobile and web apps. Ben shares how he navigated all aspects of funding, UX, developer happiness, extendable code & feature velocity to discover what he thinks is the holy grail of MVP stacks based on NodeJS. Watch and see if Ben will succeed to win your hearts and admiration or fail and die a horribly gruesome death as many others have before him!

Automated Meteor Stack Orchestration with Docker, Tutum, and Github

Jeremy Shimko

Learn how to build a fully containerized and highly scalable Meteor stack with a MongoDB replica set, HA Proxy load balancer(s), and a completely automated build process. We’ll discuss how a simple git push can trigger your app to build and deploy and we’ll see how scaling up and down can be as easy as adjusting a slider that changes your app container count.

Dux in the Channel - Using Redux, React, Phoenix and Elixir together

James Smith

Redux and React allows you to build client applications with a single direction data flow using powerful tools like time traveling debugging. All of this comes from how Redux embraces functional programming practices. It also uses patterns that are found in Erlang/OTP and by extension Elixir. Which is why using Elixir and Phoenix as your backend system along with it makes a lot of sense. In my talk I will go over what a React / Redux app wired up to Phoenix using channels looks like and how to leveage both to build exciting interactive applications.
